# AI-Powered UGC Ad Prompt System (D2C Skincare)

Today’s digital brands don’t win with polished, expensive commercials. They win with **UGC-style content**—ads that look real, raw, relatable, and human. 

This repository contains an AI-powered framework engineered to instantly generate high-converting, scroll-stopping User Generated Content (UGC) scripts and hooks. By utilizing specific persona-assigning and framework constraints, this system mimics the exact workflow of top-tier performance marketing agencies.

---

## 🎯 Project Objective
* **Automate Ideation:** Eliminate creative block by instantly generating multiple variations of hooks and full-length scripts.
* **Direct-Response Frameworks:** Every asset built follows strict psychological conversion funnels: **Hook → Agitate the Problem → Solution/Demo → Social Proof → Call-To-Action (CTA)**.
* **Platform Native:** Formatted explicitly for fast-paced vertical video formats like Instagram Reels, TikTok, and YouTube Shorts.

---

## 📂 Repository Structure

*   📂 **`prompts/`** (The Engineered AI Framework)
    *   `hooks_prompt.md`: Structured to generate 5 distinct psychological hooks (FOMO, Curiosity, etc.) using a data-table layout.
    *   `script_prompt.md`: A direct-response blueprint that dictates pacing, B-Roll visual cues, and voiceovers.
*   📂 **`outputs/`** (The Case Study Deliverables)
    *   `generated_hooks.md`: Clean, actionable scroll-stoppers mapped against visual actions.
    *   `ugc_script.md`: A fully produced 30-second multi-column script ready for a creator to shoot.

---

## 🏷️ Case Study Focus: "GlowHydra"
To demonstrate the efficiency of this prompt system, the output files are structured for a fictional D2C skincare brand called **GlowHydra**—a vegan, caffeine-infused under-eye cream targeting Gen Z and Millennials who face sleep deprivation and puffiness.

### Core Conversion Features Included:
1. **The 3-Second Scroll-Stopper:** Addresses a highly visual pain point right away ("bruise-like under-eyes", "Victorian ghost").
2. **The "Pattern Interrupt" Visual:** Physically throwing away traditional, non-working remedies (cucumbers, spoons) to capture attention.
3. **The Micro-Demonstration:** Explicit callouts for showing product texture and application on camera, building immediate consumer trust.

---

## 🛠️ How to Reuse This System
You can repurpose this system for **any** product (SaaS tools, local businesses, gadgets) by tweaking the master prompts in the `prompts/` directory:
1. Copy the raw text from `prompts/script_prompt.md`.
2. Swap out the **Product**, **Key Benefits**, and **Target Audience** variables with your own business details.
3. Run it through an LLM (Gemini, ChatGPT, or Claude) to get a completely tailored content pack in seconds.
